# BachelorThesis

## Requirements
- .NET Framework 4.5.2.
- Visual Studio 2017.

## Features 
- Visualization of the basic geometric objects such as point, line and circle.
- Finding of the intersections of two basic objects and working with them.
- Custom macros.
